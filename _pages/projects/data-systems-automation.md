# 🌍 Multi-Country Data Systems Automation

## Overview

Designed and implemented automated data management workflows supporting humanitarian and development programs across multiple countries.

## Challenge

Country offices were collecting large volumes of program and survey data through multiple platforms. Data extraction, transformation, cleaning, and reporting were largely manual.

## My Role

- Designed automated data pipelines
- Built data transformation workflows
- Developed standardized reporting structures
- Supported country teams during implementation
- Conducted user testing and troubleshooting
- Created documentation and user guides

## Technologies

- Microsoft Fabric
- Data Factory
- Power Query
- SQL
- ONA
- Excel

## Results

- Reduced reporting preparation time by more than seven days
- Improved consistency of country reporting
- Reduced manual data cleaning requirements
- Increased accessibility of program data for decision-makers
- Strengthened data governance across multiple country offices

## Skills Demonstrated

**Data Engineering • Data Governance • Automation • Business Intelligence • Stakeholder Management**

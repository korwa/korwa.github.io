# 🦟 ITN Distribution Dashboard

## Overview

Developed an interactive dashboard to monitor insecticide-treated net distribution activities and identify gaps in referral and redemption rates.

## Challenge

Program teams required timely visibility into distribution performance to improve coverage, monitor implementation progress and increase utilization.

## My Role

- Designed dashboard requirements
- Built data models
- Developed visualizations
- Created monitoring indicators
- Conducted data quality reviews

## Technologies

- Power BI
- Excel
- SQL

## Key Insights

### Households Without ITNs

2,521 households identified as lacking access to insecticide-treated nets.

### Referrals Made

2,025 households referred for ITN redemption.

### Redemption

822 households successfully redeemed ITNs.

### Gap Analysis

Approximately 20% of households were not referred and 59% of referrals were not redeemed.

## Impact

- Enabled targeted follow-up activities
- Improved monitoring of distribution performance
- Supported data-driven decision-making for public health interventions

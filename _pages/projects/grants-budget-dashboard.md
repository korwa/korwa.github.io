# 💰 Grants & Budget Pipeline Dashboard

## Overview

Developed a Power BI dashboard to support grant management, proposal tracking, funding pipeline monitoring and budget oversight.

## Challenge

Leadership needed a centralized and interactive solution to monitor proposal status, funding opportunities, budget allocations and pipeline performance.

## My Role

- Gathered reporting requirements
- Designed data models
- Developed Power BI dashboards
- Created DAX measures and KPIs
- Supported dashboard adoption and user training

## Technologies

- Power BI
- Power Query
- Excel
- DAX

## Dashboard Features

- Proposal tracking
- Funding pipeline monitoring
- Budget utilization analysis
- Award status reporting
- Executive summary KPIs
- Interactive filtering and drill-through

## Impact

- Improved visibility into funding opportunities
- Reduced reliance on manual reporting
- Supported evidence-based planning
- Enhanced executive oversight of grant performance

## Skills Demonstrated

**Business Intelligence • Dashboard Design • Data Modeling • Executive Reporting**
